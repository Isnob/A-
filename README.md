# Pathfinding Visualizer

Визуализатор поиска пути на PyQt6 с поддержкой A*, BFS и Дейкстры. Лабиринты хранятся в `config/config.yaml`, их можно генерировать случайно, импортировать из другого YAML и проигрывать пошагово в GUI или в консоли.

## Возможности
- GUI: выбор лабиринта, генерация/удаление, импорт YAML, выбор алгоритма, старт/пауза/шаг/сброс, настройка скорости.
- CLI: генерация лабиринта, удаление, пошаговая консольная симуляция с подсветкой frontier/visited.
- Алгоритмы: A* (евклидова клеточная версия с эвристикой Чебышева), BFS, Дейкстра.

## Запуск GUI
```bash
python -m src.gui.app
```
По умолчанию используется `config/config.yaml`. В окне слева — панель управления, справа — отрисовка лабиринта и состояния алгоритма.

## CLI
- Генерация случайного лабиринта и запись в `config/config.yaml`:
  ```bash
  python src/main.py generate maze_name --width 50 --height 20 --density 0.45 --start-x 1 --start-y 1 --finish-x 40 --finish-y 18
  ```
- Консольная симуляция:
  ```bash
  python src/main.py run_cli_simulation maze_name astar 0.2
  ```
  Поддерживаемые алгоритмы: `astar`, `bfs`, `dijkstra`. Последний аргумент — задержка между шагами (секунды).
- Удаление лабиринта:
  ```bash
  python src/main.py delete maze_name
  ```

## Формат `config/config.yaml`
Словарь вида `имя_лабиринта: [строки]`, где:
- `S` — старт, `F` — финиш.
- `1` — стена, `0` — проходимая клетка.
- Внешняя рамка обычно заполняется стенами.

Пример:
```yaml
demo:
  - "11111"
  - "1S0F1"
  - "11111"
```
