# Pathfinding Visualizer (skeleton)

Этот репозиторий содержит только файловую структуру проекта из `docs/TZ.md`. Реализацию логики, GUI и алгоритмов пользователь добавит самостоятельно.

## Формат конфигурации

Генератор лабиринтов записывает каждый ряд в `config/config.yaml` одной строкой (например `111101S001`). YAML остаётся компактным и легко правится вручную; при загрузке строки автоматически разбираются на клетки.
